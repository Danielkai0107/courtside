# 首頁顯示範圍擴大

## 修改內容

### 1. 擴大賽事顯示範圍 ⭐

**修改前**：

```typescript
status: ["ONGOING", "live"]; // 只顯示進行中的賽事
```

**修改後**：

```typescript
status: [
  "REGISTRATION_CLOSED", // 已截止報名（可能已發布賽程）
  "ONGOING", // 進行中
  "live", // 舊狀態（進行中）
  "scheduled", // 舊狀態（已發布）
];
```

**效果**：

- 只要賽事已發布賽程就會顯示
- 不依賴主辦方手動改狀態為 ONGOING

### 2. 擴大比賽顯示範圍 ⭐

**修改前**：

```typescript
// 只顯示進行中的
m.status === "IN_PROGRESS" || m.status === "live";
```

**修改後**：

```typescript
// 顯示即將開始和進行中的
m.status === "SCHEDULED" || // 已排程（即將開始）
  m.status === "IN_PROGRESS" || // 進行中
  m.status === "live" || // 舊狀態
  m.status === "scheduled";
```

**效果**：

- 選手可以看到即將開始的比賽
- 知道什麼時候要到場

### 3. 更新標題和提示 ⭐

**標題**：

```
首頁 → 🏠 賽事直播
```

**空狀態提示**：

```
目前沒有進行中的賽事
  ↓
目前沒有進行中或即將開始的賽事
已發布賽程的賽事會顯示在這裡
```

### 何時會顯示

```

  1. 賽事狀態 = ONGOING
  2. 比賽狀態 = IN_PROGRESS

→ 兩個條件都要滿足
```

**現在**：

```
只要：
  1. 賽事狀態 = REGISTRATION_CLOSED 或 ONGOING
  2. 比賽狀態 = SCHEDULED 或 IN_PROGRESS

→ 更容易顯示
```
